== The Read Eval Print Loop (REPL)

[source, console]
$ lein repl

We type in an expression:

[source, clojure]
(+ 1 2)

Clojure returns a result:

[source, clojure]
=> 3

NOTE: `=>` means evaluates to


[NOTE.speaker]
--
* evaluates
* compiles
* change program definitions on the fly
* try things
* answer questions
--
